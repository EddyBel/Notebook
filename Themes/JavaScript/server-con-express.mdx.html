Como hacer servidor con nodejs

Crear un archivo pakage.json para iniciar con el comando.

Bash
npm init

Instalar la libreria de express con el comando.

Bash
npm install express --save

Crear el archivo de script llamado como guste, ejemplo index.js. En el se importara la libreria y se ejecutara.

JavaScript
const express = require("express"); //Importar la libreria
const app = express(); //Ejecutar la libreria

Especificar el archivo que se enviara con su funcion get.

JavaScript
app.get("/", (req, res) => {
  res.send("Hello World"); //Enviaremos un texto plano 'Hello World'
});

Definir el puerto donde se ejecutara el servidor.

JavaScript
app.listen(9090); //Escuchara en el puerto 9090

Archivo completo index.js
